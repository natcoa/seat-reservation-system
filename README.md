# Airline Seat Reservation System (Java)

A command-line application that simulates an airline seat booking system using object-oriented programming.

## Features
- View seat availability in a 2D seating layout
- Reserve seats by row and letter
- Search for passengers by name
- Display passenger list with seat details

## Design
- Uses a 2D array to represent plane seating
- Custom Seat class to manage passenger data and seat type
- Implements encapsulation, constructors, and enums

## Technologies
- Java
- Object-Oriented Programming (OOP)
